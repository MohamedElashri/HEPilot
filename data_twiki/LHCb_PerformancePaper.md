Editorial Details =================

SVN Repostiory \-\-\-\-\-\-\-\-\-\-\-\-\--

A latex template for the paper has been set up in
\[LHCbDocs\](https://twiki.cern.ch/twiki/bin/view/LHCb/LHCbDocs).

You can download a working copy of the paper using the command

svn co \$DLHCB/Publications/PubDrafts/2012/LHCbPerformancePaper

You can just browse the repository on the web using \[this
link\![\](./LHCbPerformancePaper \_ LHCb \_
TWiki_files/external-link.gif)\](https://svnweb.cern.ch/cern/wsvn/lhcbdocs/Publications/PubDrafts/2012/LHCbPerformancePaper).

You will find the repository has the normal
\[LHCbDocs\](https://twiki.cern.ch/twiki/bin/view/LHCb/LHCbDocs) setup.
A \`latest\` directory contains the working document, which is where
edits should be made. A \`drafts\` directory exists for any future
tagged versions we wish to save.

The \`latest\` directory contains three more. A \`latex\` directory for
the latex source, a \`figs\` directory for figures and a \`reference\`
directory for any reference information. For instance reference/IJMPA
contains the files downloaded from the \[IJMPA web
site\![\](./LHCbPerformancePaper \_ LHCb \_
TWiki_files/external-link.gif)\](http://www.worldscinet.com/style_files/ijmpa/139-readme_2e.shtml)
which includes guidelines for the journal and example tex files.

Authors should aim to keep the SVN repository up to date with their own
changes, and also to regularly update their working area from SVN, to
pick up changes from others. If you allow SVN and your working copies to
diverge too far, it can be problematic to merge things back into sync.

To update you area, just run

svn update

and to commit changes

svn commit -m \"Some meaningful comment\"

After making any major commits, please send an email to
\`lhcb-performance-paper-authors\` so the other authors can update.

\### Building the document

A Makefile has been added to the \`latex\` directory that will build the
PDF document from the latex source. Just take \`make\`. If you wish to
do a clean rebuild, run \`make clean\` first.

Editting Guidelines \-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\--

When editting your sections please pay attention to both the LHCb and
IJMPA guidelines. The IJMPA guidelines can be found in the
\`reference/IJMPA\` directory. For LHCb guidelines, please familiarise
yourself with the guidelines as can be found from the \[LHCb editorial
web pages\![\](./LHCbPerformancePaper \_ LHCb \_
TWiki_files/external-link.gif)\](http://lhcb.web.cern.ch/lhcb/edboard/Version2010/editorial-board.html).

For convenience, I have also uploaded PDFs of the
\[IJMPA\](https://twiki.cern.ch/twiki/pub/LHCb/LHCbPerformancePaper/ws-ijmpa.pdf)
and
\[LHCb\](https://twiki.cern.ch/twiki/pub/LHCb/LHCbPerformancePaper/LHCb-Guidelines.pdf)
guidelines to this twiki.

Please can I urge the authors to abide by these guidelines from the
start, as it will make things easier in the long run.
